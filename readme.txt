These steps to run app in Windows 
# Create a new environment

Step 1 : open Anaconda prompt
                       
Step 2 : Create a new environment	
command : conda create --name my_project_env

Step 3 : Activate your environment
                
Command: conda activate my_project_env

Step 4 : Install requirements.txt in the newly created environment
            
	a). Navigate to your folder location on anaconda prompt/teminal
. 
	    For me ( C:\Users\vedav\work\DataScience_iNeuron\MLDL_Project\Project_Final)
                 
	    For you - you folderName
            
	b). Check if we have requirements.txt or not in the current directory
	Command:ls
Step 5 : Installation and setup is done:
  
Command: pip install -r  requirements.txt
Step 6 : Run the app
               
Command: python app.py

Step7:after running, you will get the URL to see the app
http://127.0.0.1:5000/
